8. tolong dibantubesok mau dikumpul


Diana memesan 2 porsi rendang dan 2 porsi telur, jumlah uang yang harus dibayar sebesar Rp10.000

Penyelesaian

misal,

x = porsi ayam

y = porsi rendang

z = porsi telur

4x + 2y + 3z = 26.000 ... pers 1

3x + 3y + z = 21.500 ... pers 2

3x + z = 12.500 ... pers 3

4x + 2y + 3z = 26.000 | x 1|

3x + 3y + z = 21.500  | x 3|

4x + 2y + 3z = 26.000

9x + 9y + 3z = 64.500

______________ -

-5x - 7y = -38.000 ... pers 4

3x + z = 12.500

3x = -z + 12.500 ... pers 5

Substitusi pers. 5 ke pers. 2

3x + 3y + z = 21.500

-z + 12.500 + 3y + z = 21.500

3y = 21.500 - 12.500

3y = 9.000

y = 3.000

Substitusi nilai y ke salah satu persamaan.

-5x - 7(3.000) = -38.500

-5x - 21.000 = -38.500

-5x = -38.500 + 21.000

-5x = -17.500

x = 3.500

Substitusi nilai x dan y untuk mendapatkan z

3x + 3y + z = 21.500

3(3.500) + 3(3.000) + z = 21.500

10.500 + 9.000 + z = 21.500

19.500 + z = 21.500

z = 21.500 - 19.500

z = 2.000

(x , y , z) = ( 3.500 , 3.000 , 2.000)

Memesan dua porsi rendang dan dua porsi telur

= 2x + 2y

= 2(3.000) + 2(2.000)

= 6.000 + 4.000

= 10.000

KesimpulanJadi , jika Diana memesan 2 porsi rendang dan 2 porsi telur, jumlah uang yang harus dibayar sebesar Rp10.000
