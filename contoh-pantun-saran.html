Pantun adalah bentuk puisi Indonesia ( Melayu ) yang tiap baitnya terdiri atas empat ( 4 ) baris✔️

Ciri-ciri pantun ada empat ( 4 ), yaitu :

1. Pantun bersajak a - b - a - b✔️2. Satu ( 1 ) bait terdiri atas empat ( 4 ) baris✔️3. Tiap hari terdiri atas delapan ( 8 ) sampai duabelas ( 12 ) suku kata✔️4. Dua ( 2 ) baris pertama adalah sampiran dan dua ( 2 ) baris berikutnya adalah isi pantun✔️
