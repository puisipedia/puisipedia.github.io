= 36 - 6
= 30 Cm

Jawab

<=> Panjang lukisan / Panjang Bingkai = Lebar Lukisan / Lebar Bingkai

<=> x / 60 = 30 / 36

<=> 36 . x = 60 x 30

<=> 36x = 1800

<=> x = 1800 / 36

<=> x = 50


<=> x = { 60 - { 3 + 50 }

<=> x = 60 - 53

<=> x = 7 Cm



Jadi Sisa Pada Bagian Bawah Lukisan Tersebut Adalah 7 Cm
