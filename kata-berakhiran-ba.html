Penjelasan Soal

Bilangan bulat adalah bilangan yang unsurnya terdiri atas bilangan bulat negatif, bilangan nol, dan bilangan positif.

-a = Negatif0 = nol+a / a = positif

Didalam mengoperasikan campuran bilangan bulat, tentunya kita tahu kata penjumlahan, pengurangan, perkalian, dan pembagian. Apa saja sih itu? perhatian penjelasan berikut ini :

1. Penjumlahan Bilangan Bulat

Penjumlahan adalah operasi aritmatika pertambahan suatu bilangnya yang dilambangkan dengan simbol " + " ( plus ). Sifat-sifat penjumlahan sebagai berikut :

Sifat Komutatif (Pertukaran)

[tex] \boxed{a + b = b + a} [/tex]

Sifat Assosiatif (Pengelompokan)

[tex] \boxed{(a + b) + c = a + (b - c)} [/tex]

Unsur identitas/unsur netral

[tex] \boxed{a + 0 = 0 + a = a} [/tex]

Sifat tertutup

-a + b = c

bilbul + bilbul = bilbul

Jadi semua bilangan bulat jika ditambahkan dengan bilangan bulat hasilnya pasti bilangan bulat juga. Sifat demikian dinamakann sifat tertutup.

Pengoperasian penjumlahan bilbul

a + b = c-a + (-b) = - (a + b)

-a + b = b - a, untuk b > a-a + b = - (a - b), untuk b < a

2. Pengurangan

Pengurangan adalah operasi aritmatika dengan mengurangi suatu bilangan. Pengurangan dilambangkan dengan simbol " - " ( minus ).

Pengoperasian pengurangan bilbul

a - b = a + (-b)-a - (-b) = -a + b

-a - b = - (a + b)a - (-b) = a + b

3. Perkalian

Perkalian adalah operasi penjumlahan secara berulang menurut bilangan yang dikalikannya. Perkalian dilambangkan dengan simbol " × ". Sifat-sifatnya perkalian dalam bilangan bulat sebagai berikut :

Sifat Komutatif :

[tex] \boxed{a \times b = b \times a} [/tex]

Sifat Assosiatif

[tex] \boxed{(a \times b) \times c = a \times (b \times c)} [/tex]

Sifat Distributif

[tex] \boxed{a (b + c) = (a \times b) + (a \times c)} [/tex]

Pengoperasian perkalian bilbul

a × b = c-a × b = -ca × (-b) = -c

Positif × Positif = PositifNegatif × Positif = NegatifPositif × Negatif = Negatif

4. Pembagian

Pembagian adalah operasi pengurangan secara berulang menurut bilangan yang dibaginya. Pembagian dilambangkan dengan simbol " ÷ / : ".

Pengoperasian pembagian bilbul

a ÷ b = c-a ÷ b = -ca ÷ (-b) = -c

Positif ÷ Positif = PositifNegatif ÷ Positif = NegatifPositif ÷ Negatif = Negatif

5. Campuran

Dalam operasi hitung campuran ini, melibatkan antara dua atau lebih operasi hitung bilangan penjumlahan, pengurangan, perkalian, maupun pembagian.

1. Penjumlahan

Penjumlahan adalah operasi matematika yang dimana terdapat simbol tambah atau "+" yang memisahkan antar bilangan agar dapat dinamakan penjumlahan dan dioperasikan pertambahan

2. Pengurangan

Pengurangan adalah operasi matematika dimana operasinya disimbolkan dengan simbol kurang atau "-" dan dioperasikan dengan cara mengurangkan suatu bilangan

3. Perkalian

Perkalian adalah operasi matematika dasar yang melibatkan penjumlahan berulang dan disimbolkan dengan simbol kali atau "×"

4. Pembagian

Seperti hal-nya perkalian, namun pembagian adalah kebalikan dari perkalian. Pembagian ini yaitu pengurangan berulang yang disimbolkan dengan simbol bagi atau ":" atau "÷" atau juga per "/"

Dalam operasi hitung campuran, dahulukan operasi sebagai berikut :

Tentukan Perpangkatan Tersebut!

Pendahuluan

Bilangan Berpangkat merupakan bentuk sederhana dari perkalian berulang. Bilangan Berpangkat dinyatakan sebagai bilangan yang dikalikan dengan bilangan itu sendiri secara berulang.

Setiap perkalian berulang dapat dituliskan ringkas menggunakan notasi bilangan berpangkat.

Contoh :

2 × 2 = 2² ( Dibaca 2 Pangkat 2 ) 3 × 3 × 3 = 3³ ( Dibaca 3 Pangkat 3 ) 4 × 4 × 4 × 4 =  ( Dibaca 4 Pangkat 4 )

Berdasarkan contoh tersebut, maka bilangan berpangkat dimana :

a = Bilangan ril n = Bilangan bulat positif[tex]\bf a^n = a \times a \times a \times a \times a \ ... \times a[/tex]

Sebanyak n Faktor

✏ ️Contoh Bilangan Perangkatan 2 Yaitu : 4 × 4 = 4².

Dibaca empat pangkat lima. Kenapa ? Karena 4 nya itu dikali kan sebanyak lima kali. Dan jadi lah Dibaca empat pangkat dua atau empat kuadrat [4²].

✏️ Contoh Bilangan Perpangkatan 3 Yaitu : 2 × 2 × 2 = 2³.

Dibaca dua pangkat tiga. Kenapa ? Karena 2 nya itu dikali kan sebanyak tiga kali. Dan jadilah Dibaca dua pangkat tiga [2³].

✏️ Akar Pangkat Dua Yaitu : n × n = g, Maka, √g = n.

√g dibaca akar pangkat dua atau akar kuadrat dari g.

✏️ Akar Pangkat Tiga Yaitu :

Hasil akar pangkat tiga dari suatu bilangan ialah bilangan yang jika dipangkatkan tiga menghasilkan bilangan di dalam akar tersebut.

³√125.

Dibaca Akar Pangkat Tiga dari bilangan 125 tersebut.

31. QUIS#50perumutasi dari kata "BA"​


Pendahuluan :

( Materi soal )

Kaidah Pencacahan atau dapat dikenal sebagai ( Counting Rules ) Merupakan suatu Cara Dalam Matematika Untuk Menghitung Jumlah keseluruhan Kemungkinan yang bisa terjadi dalam suatu percobaan tertentu .

———

Kaidah Pencacahan biasanya dipakai untuk Menghitung Suatu Huruf / kata yang Bisa Dibagi Dalam suatu Kalimat , Setelah Itu Kita disuruh Untuk Mencari Berapa banyak susunan Kata didalam nya , Selain itu Didalam Kaidah Pencacahan Terdapat Beberapa Unsur² didalam nya Diantara nya :

1.Faktorial

2.Permutasi

3.Kombinasi

4.Aturan Pengisian Tempat

===============================» Pembahasan :

1.Faktorial

Faktorial ialah suatu bilangan Asli yang dilambangkan dengan Huruf (N) yang dimana Faktorial sendiri merupakan Perkalian Berurutan yang diawali dengan Angka -1 sampai Bilangan asli .

Rumus :

N ! = n × ( n − 1 ) × ( n − 2 ) × ( n − 3 ) .... × 3 × 2 × 1

keterangan :

N = Merupakan Bilangan Asli Yang Difaktorialkan

———2.Permutasi

Permutasi Ialah suatu susunan/urutan Objek yang Berbeda Dari sebelum nya.

—Rumus Permutasi :

• Untuk Permutasi apabila Didalam Soal Tidak Terdapat Unsur ganda Bisa Menggunakan Rumus :

P = N !

• Apabila Terdapat Unsur ganda maka akan Menggunakan Rumus :

P = N ! / K !

Keterangan :

P = Unsur ganda

N = Total Huruf

K = Huruf Ganda

! = Faktorial

———3.Kombinasi

Kombinasi Sendiri Merupakan suatu susunan berurutan dari suatu himpunan tanpa memperhatikan pola urutan

—Rumus Kombinasi :

[tex]c \frac{n}{r} = \frac{n}{r ! ( n - r )} [/tex]

———4.Aturan Pengisian Tempat

Aturan Pengisian Tempar Digunakan untuk menentukan banyak cara dalam suatu Kata untuk menempati sebuah tempat .

====================================

38. Q. [ 5/5 ]78³ + 77² - 5² = ....------Quiz terakhir.BA bulan depan.​


78³ + 77² - 5²

= 78³ + (77 - 5)(77 + 5)

= 78³ + (72)(82)

= 78³ + 5.904

= (78 × 78 × 78) + 5.904

= (6.084 × 78) + 5.904

= 474.552 + 5.904

= 480.456

:)Penyelesaian:

78³ + 77² - 5²

= (78 × 78 × 78) + 77² - 5²

= (6.084 × 78) + 77² - 5²

= 474.552 + (77 × 77) - 5²

= 474.552 + 5.926 - (5 × 5)

= (474.552 + 5.926) - 25

= 480.478 - 25

= 480.453

Maka, hasilnya adalah 480.453.
