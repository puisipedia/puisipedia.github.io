22. tolong bantuin ade/kkdikumpulin nnti malemjgn ngasalyg jawab cuma buat ambil poin gw, gw g bakal ikhlas​


Jawab:

1. Busur adalah kurva lengkung yang berimpit dengan lingkaran.

2. Juring, dan tembereng.

3. Diameter.

4. a) Diketahui : r : 28 cm

       Ditanya : K

       Jawab : K= π x 2 x r

                   = 22/7 x 2 x 28

                   = 176 cm

   b) Diketahui : r : 8 cm

       Ditanya : K

       Jawab : K= π x 2 x r

                   = 22/7 x 2 x 8

                   = 352/7 cm

5. a) Diketahui : d : 42 cm

       Ditanya : L

       Jawab : L= π x r²

                   = 22/7 x (21)²

                   = 22/7 x 21 x 21

                   = 1386 cm²

b) Diketahui : d : 18 cm

       Ditanya : L

       Jawab : L= π x r²

                   = 3,14 x (9)²

                   = 3,14 x 9 x 9

                   = 254,34 cm²
