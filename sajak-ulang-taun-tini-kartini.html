14. 1. Dari Data ulangan ips tini SBB: 78.80.82.90 tentukan rata² nilai ulangan tini Jawap​


Jawab:

Rata - rata nilai ulangan tini adalah 82.5

Penjelasan dengan langkah-langkah:

Jumlah nilai ulangan IPS Tini = 78 + 80 + 82 + 90 = 330

Jumlah data = 4 (karena terdapat 4 nilai ulangan)

Rata-rata nilai ulangan IPS Tini = Jumlah nilai ulangan / Jumlah data

= 330 / 4

= 82.5

Jadi, rata-rata nilai ulangan IPS Tini adalah 82.5.

27. Nilai ulangan harian Tini adalah 75,80,90,65,dan70.Rata-rata nilai ulangan harian Tini adalah....


Diketahui:

Nilai ulangan rini = 75,80,90,65,dan70

Banyak data =5

Ditanya:

Rata-rata nilai ulangan rini

Jawab:

[tex]=\frac{75+80+90+65+70}{5}[/tex]

[tex]=\frac{380}{5}[/tex]

=76

Jadi nilai rata rata ulangan rini adalah 76
