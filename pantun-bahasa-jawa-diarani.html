9. Apa yang diarani drama menurut bahasa jawa


Pengertian :

Drama yaiku salah sawijining jenis karya sastra ingkang nggambaraken kedadeyanipun utawa realita uripe menungsa lan diperagakaken dening tokoh kang anduwe watak, uga tingkah laku. Serta wonten dialog kanggo dipentasaken.

Ciri-ciri :Drama diceritakaken lewat dialog utawa pocakapan saking para tokoh.Nduweni tokoh kang diperanaken lan anduwe watak utawa karaktere beda-beda.Drama ditujukake kanggo hiburan masyarakat.Drama dipentasaken ing nduwur panggung lan dipunlengkapi peralatan pendukung lakuning drama.Sajroning drama wonten konflik ingkang dadi intining drama.

Struktur :

Drama nduweni struktur supados penonton saged nikmati lakunipun drama ingkang dipentasaken.

Babak (episode) yaiku rangkuman peristiwa utawa kedadeyan ing panggonan ngangge urutan waktu tartamtu.Adegan yaiku owahaning kedadeyan. Saged dipun tandai ngangge pergantian tokoh utawa setting panggonan lan wektu.Dialog yaiku pocakapan tokoh siji lan sijine kanggo nguripaken cerita.Prolog yaiku gambaran umum ngenani lakuning drama ingkang dilakoni.Epilog yaiku kesimpulanipun saking drama ingkang dipentasaken.Unsur- unsur :

Unsur-unsur drama digunakake kanggo mbangun lakuning cerita ingkang kasusun saking :

Tema : ide utama utawa inti saking drama.Tokoh : peraga kang njalanaken peran sajroning drama.Watak : karakteripun utawa sifat saking tokoh.Alur : kronologi utawa lakuning drama.Latar : papan panggonan/ wektu.Amanat : pesanipun pengarang marang penonton saking peran kang dilakokake dening tokoh.

30. Opo kang di arani tembaong camboran dalam bahasa jawa


Tembung camboran tegese loro utawa sing dirangkep dadi siji.

Tuladha: meja tulis, buku gambar, balung kuwuk, sayuk rukun, narapati.

Warna-warnane tembung camboran.

Manut wujudeCamboran tugel, yaiku tembung camboran kang dumadi saka tembung lorosing dirangkep dadi siji kanthi ngurangi jumlah wandane.

Wujude camboran tugel

1.Tembung garba (sandi)

Tuladha: Nara + pati = narpati

Prapta + ing = prapteng

Idu + abang = dubang

Iya + iku = yeku

Sura + ing = sureng

Parama + iswari = prameswarI

2. Kerata basa

Tuladha: Digugu + ditiru = guru

Balung + kulit = lunglit

Gawe + kadang = wedang

Kaku + cempluk = kupluk

B. Camboran wutuh, yaiku tembung loro sing dirangkep dadi siji nanging ora nganggo ngurangi jumlah wandane.

Wujude camboran wutuh:

1. Rerangken determinatif (DM)

Tuladha : Meja tulis

Lemari kaca

Pager bata

Pitik walik

Buku gambar

2. Baliswara (rerangken MD)

Tuladha : Pancasila

Bimaputra

Wijaya kusuma

Mahasiswa

Dwiwarna

3. Tembung saroja

Tuladha : Sanak kadang

Wa sumelang

Papa cintraka

Japa mantra

Lara lapa

4. Yogyaswara

Tuladha : Dewa-dewi

Siswa-siswi

Putra- putri

Bathara- bathari

Pemudha- pemudhi

5. Tembung kosok balen (Antonim)

Tuladha : Gedhe x cilik

Enom x tuwa

Lanang x wadon

Mangkat x mulih

Dawa x cekak

Dhuwur x cendhek

2. Manut kekarepane

A. Cmboran tunggal, yaiku tembung loro sing dirangkep dadi siji, nanging tembung siji lan sijine wis ora kena dipisah-pisah maneh lan wis nduweni teges anyar.

Upamane balung iku perangane awak utawa kewan sing atos dhewe. Kuwuk iku kucing alasan ora ana sing ndarbeni. Banjur dirangkep dadi “balung kuwuk” sing tegese criping pohung.

Tuladha: Randha royal

Semar mendem

Nagasari

Gandarukem

Daramuluk

B. Camboran wudar, yaiku tembung camboran sing tembung siji lan sijine isih nduwe teges dhewe-dhewe.

Tuladha: Meja tulis

Pelem golek

Wayang kulit

Piring beling
