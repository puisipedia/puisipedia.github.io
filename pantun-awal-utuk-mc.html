18. Momentum suatu elektron awalnya sama dengan mc. Untuk memperkecil panjang gelombang de Broglie elektron ini agar menjadi setengah dari semula maka energi total elektron tersebut harus diubah menjadi .... mc^2. tolong jawab pakai cara yang lengkap yaa


*Energi kinetik (EK)*
EK = ½ m.v²

dalam soal, untuk elektron
EK = ½ m.c²

c elektron = kecepatan elektron (m/s)


*Panjang gelombang de Broglie*
λ = h / p

h = kontanta planck
h = 6,6 x 10^-34 Js
p = momentum (kg m/s)

Dalam soal diketahui momentum elektron = mc

maka,
λ = h / (mc)
c = h / (mλ)

Kita subsitusi ke EK
EK = ½ m c²
EK = ½ m (h / mλ)²
EK = ½ m h² / (m².λ²)
EK = ½ h² / (m.λ²)

terlihat EK berbanding terbalik dengan λ²
EK2 / EK1 = (λ1 / λ2)²


Diketahui :
λ1 = λ
EK1 = ½mc²

panjang gelombang de Broglie diperkecil menjadi ½ dari semula:
λ2 = ½λ

Ditanyakan : EK2

Jawab :
EK2 / EK1 = (λ1 / λ2)²
EK2 / EK1 = (λ / ½λ)²
EK2 / EK1 = 4
EK2 = 4 EK1
EK2 = 4 (½mc²)
EK2 = 2mc²


Kelas : 12
Mapel : Fisika
Kategori : Relativitas
Kata Kunci : panjang gelombang de Broglie, Energi kinetik, Energi Total
Kode : 12.6.9
#backtoschoolcampaign

• Dualisme
• TRK

p = mₒ c
(mₒ = massa diam)
λ' = ½ λ → E' = __?

Panjang gelombang de-broglie
λ = h / p
λ berbanding terbalik dengan momentum

λ' / λ = (h/p') / (h/p)
λ' / λ = p / p'
p' = (λ / λ') p
p' = (λ / ½λ) p
p' = 2 p
p' = 2 mₒ c

Energi total
E'² = (p' c)² + (mₒ c²)²
E'² = (2 mₒ c • c)² + (mₒ c²)²
E'² = 4 (mₒ c²)² + (mₒ c²)²
E'² = 5 (mₒ c²)²
E' = √5 mₒ c². ✔️

33. mengapa kita harus mendahulukan kewajiban sebelum menutuk hak ? ​


Kita harus mendahulukan kewajiban sebelum menuntut hak, karena kewajiban merupakan sebuah keharusan yang wajib terlaksana terlebih dahulu, agar kita dapat memperoleh suatu hak. Maka kewajiban harus kita dahulukan pelaksanaanya.

PEMBAHASAN:

Hak adalah segala sesuatu yang sudah menjadi keharusan untuk kita dapatkan, sedangkan kewajiban merupakan segala sesuatu yang sudah menjadi keharusan untuk kita laksanakan. Kewajiban merupakan sebuah alat yang dapat digunakan untuk dapat memperoleh hak. Hak dan kewajiban sangat berhubungan erat, oleh sebab itu pelaksanaan keduanya harus secara seimbang agar tidak terjadi kekacauan dalam pelaksanaanya. Contoh hak sebagai warga dimasyarakat:

Berhak menikmati lingkungan yang bersih.Berhak menikmati fasilitas umum.Berhak memeluk agama sesuai kepercayaan.Berhak dihormati.Berhak memperoleh aliran listrik.Berhak memperoleh pendidikan.Berhak mendapatkan perlindungan hukum

Contoh kewajiban sebagai warga masyarakat:

Wajib ikut serta menjaga kebersihan lingkungan.Wajib merawat serta menjaga fasilitas umum.Wajib ikut serta menjaga keamanan dan kenyamanan lingkungan.Wajib membayar pajak tepat waktu.Wajib ikut serta mengikuti kegiatan-kegiatan sosial dimasyarakat.Wajib mematuhi segala peraturan-peraturan yang berlaku di lingkungan masyarakat.

Dalam pelaksanaan hak dan kewajiban, kita harus mendahulukan kewajiban terlebih dahulu baru kemudian kita dapat menuntut hak. Setelah kewajiban terlaksana, dan hak kita sudah terpenuhi maka itu artinya pelaksanaan hak dan kewajiban dapat dikatakan sudah seimbang.

40. Momentum suatu elektron awalnya sama dengan mc. Untuk memperkecil panjang gelombang de broglie elektron ini agar menjadi setengah dari semula maka energi total elektron tersebut harus menjadi... A. akar 2 mc^2 B. akar 3 mc^2 C. akar 4 mc^2 D. akar 5 mc^2 E. akar 6 mc^2


• Dualisme
• TRK

p = mₒ c
λ' = ½ λ → E' = __?

Panjang gelombang de-Broglie
λ = h / p
λ berbanding terbalik dengan momentum

λ' / λ = (h/p') / (h/p)
λ' / λ = p / p'
p' = (λ / λ') p
p' = (λ / ½λ) p
p' = 2 p
p' = 2 mₒ c

Energi total elektron
E'² = (p' c)² + (mₒ c²)²
E'² = (2 mₒ c • c)² + (mₒ c²)²
E'² = 4 (mₒ c²)² + (mₒ c²)²
E'² = 5 (mₒ c²)²
E' = √5 mₒ c²
E' = √5 m c² ✔️
