13. Bantu Dong Yang Jawara MTK​


dalam layang layang itu, ada 2 segitiga siku siku, maka kita hanya tinggal mengalikannya dengan 2 setelah mencari luas segitiga KOL

luas layang layang = 2 × luas KOL

luas layang layang = 2 × (alas . tinggi ÷ 2)

luas layang layang = OK × KL

cari panjang KL menggunakan phytagoras di segitiga KOL :

[tex]{KL}^{2} + {KO}^{2} = {OL }^{2} [/tex]

[tex]{KL}^{2} + {16}^{2} = {34 }^{2} [/tex]

[tex]{KL}^{2} = 900[/tex]

KL = 30

luas layang layang = OK × KL = 16 × 30

luas layang layang = 480 cm2

Mapel : Matematika

Kelas : 8

Bab : 7 ( garis singgung )

ꔚ

Penyelesaian

[tex]kl = lm[/tex]

[tex]kl = \sqrt{ol {}^{2} - ok {}^{2} } [/tex]

[tex] = \sqrt{34 {}^{2} - 16 {}^{2} } [/tex]

[tex] = \sqrt{900} [/tex]

[tex]kl = 30[/tex]

[tex]luas \: layang\: layang = 2 \times luas \: \: sgitiga[/tex]

[tex] = 2 \times \frac{1}{2} \times 30 \times 16[/tex]

[tex] = 480cm {}^{2} [/tex]

➜Jadi luas layang layang adalah 480 cm² (D)

Opsi D. 20 cm

Rumusnya

• Mencari Persekutuan dalam

[tex]d = \sqrt{s {}^{2} - (R + r) {}^{2} } [/tex]

[tex] d= [/tex]garis persekutuan dalam

[tex] s=[/tex] jarak dua titik pusat

[tex] R= [/tex]jari lingkaran 1

[tex] r= [/tex] jari lingkaran 2

Penyelesaiannya

[tex]d = \sqrt{25 {}^{2} -(9 + 6) {}^{2} } \\ d = \sqrt{625 - 225 } \\ d = \sqrt{400} \\ d = 20[/tex]

[tex]CD = 20 \: cm[/tex]

⭐♫Pendahuluan❖️☁☔⚘

Garis singgung adalah suatu garis yg terdapat pada lingkaran dan garis itu memotong pada suatu bangun datar lingkaran disatu titik tegak lurus pada jari² lingkaran pada titik singgung lingkaran.

Garis singgung ada dua macam :

Garis singgung persekutuan dalam dua lingkaranGaris singgung persekutuan luar dua lingkaran

Rumus : Garis singgung persekutuan dlm & luar

d = √p² - (R + r)²

l = √p² - (R - r²)

Keterangan :

d = Panjang garis singgung persekutuan dalam 2 lingkaranR = Jari-jari lingkaran besarr = Jari-jari lingkaran kecilp = Jarak antar pusat dua lingkaranl = Panjang garis singgung persekutuan luar 2 lingkaran

Penjelasan dengan langkah-langkah:

Diketahui :

AC = 9 cmBD = 6 cmAB = 25 cm

Ditanya :

Panjang CD adalah ... (Garis singgung persekutuan dalam 2 lingkaran)

Jawab :

CD = √AB² - (AC + BD)²

CD = 25² - (9 + 6)²

CD = 25 - 15

CD = 25² - 15²

CD = 625 - 225

CD = √400

CD = 20 cm → D

DETAIL JAWABAN :Mapel : MatematikaKelas : VIIIBab : 7 - LingkaranKata Kunci : Panjang garis singgung persekutuan dalam dua lingkaranKode Kategorisasi : 8.2.7
